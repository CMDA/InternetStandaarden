About this template

This document describes the most commonly used semantical HTML elements and is marked up with the HTML5 elements <header>, <nav>, <article>, <section> and <footer>. Next to CSS3 styling, behavior was added to the document with JavaScript.

Headers

The <h1> to <h6> elements are block-level elements used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading. Browsers automatically add some space (margin) before and after each heading element.

Paragraphs & text elements

Paragraph

The <p> element is a block level element representing a paragraph. Browsers automatically add some space (margin) before and after each <p> element

Inline text elements

Strong

The <strong> element defines important text in a section.

Em

The <em> element emphasizes text in a section.

Quote

The <blockquote> tag specifies a section that is quoted from another source or person.

Lists

Unordered list

The <ul> element defines an unordered (bulleted) list. Use the <li> tag to define list items.

  • Pellentesque
  • Vestibulum
  • Vehicula
  • Tellus
  • Euismod

Ordered list

The <ol> tag defines an ordered list. Use the <li> tag to define list items.

  1. Maecenas sed diam eget risus varius blandit sit amet non magna.
  2. Fusce dapibus, tellus ac cursus commodo, tortor mauris condimentum.
  3. Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  4. Nulla vitae elit libero, a pharetra augue.
  5. Sed posuere consectetur est at lobortis.
