Criteria practicumopdracht

  1. AVV = Aan Voorwaarden Voldaan
    Als je hier niet aan voldoet krijg je geen beoordeling/feedback
  2. Minimale criteria
    Criteria over je HTML en CSS waaraan je allemaal moet voldoen voor een voldoende
  3. Criteria voor ophoging
    Met deze criteria kan je je cijfer ophogen, mits je een voldoende hebt

A) AVV - Aan Voorwaarden Voldaan

  1. De website bestaat uit tenminste vier pagina’s.
  2. De website bevat relevante inhoud in correct Nederlands.
  3. De website is verzorgd en leesbaar opgemaakt.
  4. Wanneer de content niet zelf is geproduceerd, wordt de bron vermeld (volgens de MLA standaard).
  5. De website staat op Oege en is zonder technische problemen te bereiken.

B) Minimale Criteria

  1. HTML
    1. De HTML-code is semantisch correct, de inhoud is goed gestructureerd.
    2. De HTML-code is syntactisch correct, levert geen fouten op bij de validator.
    3. De website heeft een menu gebaseerd op unordered lists.
  2. CSS
    1. De CSS-code is syntactisch correct.
    2. De vormgeving is consistent doorgevoerd.
  3. Beiden
    1. De code voldoet aan de in de les aangereikte codingstandaarden.
    2. HTML is enkel gebruikt voor content, CSS enkel voor opmaak, (Javascript enkel voor gedrag) en is fysiek gescheiden.

C) Criteria voor ophoging zijn bijvoorbeeld

  1. Javascript / jQuery te gebruiken voor DOM-manipulatie.
  2. Veel aandacht te besteden aan vormgeving. Je website oogt professioneel.
  3. Elegant te coderen, bijvoorbeeld met behulp van een CSS-strategie.
  4. Cross-browser verschillen op te lossen met behulp van CSS
  5. Een technisch complexe lay-out te hanteren.
  6. Een responsive design te realiseren.
  1. Bedenk uit welke pagina's jouw website gaat bestaan
  2. Ga kijken naar verschillende menustijlen en laat je inspireren
    1. http://css.maxdesign.com.au/
    2. http://www.cssplay.co.uk/menus/
    3. http://www.webdesignerdepot.com/2011/11/navigation-patterns-for-ten-common-types-of-websites/
  3. Bekijk in de bron hoe deze menu's gemaakt zijn
  4. Start met het maken van je eigen menu in HTML
